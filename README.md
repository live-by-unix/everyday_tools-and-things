# everyday_tools-and-things
A collection of tools. 

# Libaries and Applications
Python Libaries: customtkinter, python-vlc, tkinter (For Linux users), matplotlib, requests,

Applications: VLC(Downloads from videolan.org), Python 3.12 (Recommended but not required), and Terminal

# Runtime

CD to the folder this project is in (OR if you use a venv for all libaries, cd to the venv first and skip to step 3)

Run the programs with python (or python3) and Enjoy!

IN a venv, you should first activate it with source venv/bin/activate (macOS and Linux) and for Windows, do the command for your desired shell. Then cd to the project folder and run with python or python3 and Enjoy! To exit venv do deactivate
